## 🚀 QUICK START (3 OPTIONS)

### OPTION A: Docker Compose (FASTEST - 1 COMMAND)
────────────────────────────────────────
cd api-security-platform
docker-compose up -d
# Wait 15 seconds...
curl http://localhost:3000/health
# ✅ READY!

### OPTION B: Manual Setup
────────────────────────────────────────
# Terminal 1: Start PostgreSQL (or use existing)
createdb api_security_platform

# Terminal 2: Start Backend
cd backend
npm install
cp .env.example .env
npm start

# Terminal 3: Start Scanner
cd scanner
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python -m uvicorn app.main:app --port 8000

# Terminal 4: Test
curl http://localhost:3000/health
curl http://localhost:8000/health

### OPTION C: Single Docker Command
────────────────────────────────────────
docker-compose up -d postgres backend scanner

## 📋 EXAMPLE USAGE

1️⃣ REGISTER USER
────────────────────────────────────────
curl -X POST http://localhost:3000/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@company.com","password":"SecurePass123"}'

2️⃣ LOGIN
────────────────────────────────────────
curl -X POST http://localhost:3000/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@company.com","password":"SecurePass123"}'

# Save TOKEN from response

3️⃣ UPLOAD POSTMAN COLLECTION
────────────────────────────────────────
curl -X POST http://localhost:3000/scans \
  -H "Authorization: Bearer $TOKEN" \
  -F "file=@my-api-collection.json"

# Scan starts! Returns scan_id

4️⃣ CHECK SCAN STATUS
────────────────────────────────────────
curl http://localhost:3000/scans/1 \
  -H "Authorization: Bearer $TOKEN"

# Status: pending → running → completed

5️⃣ VIEW RESULTS
────────────────────────────────────────
curl "http://localhost:3000/results?scanId=1" \
  -H "Authorization: Bearer $TOKEN"

# See vulnerabilities with severity, evidence, details

## 🐛 TROUBLESHOOTING

Port Already in Use
  $ lsof -i :3000
  $ kill -9 <PID>

Database Connection Error
  $ psql -U postgres -d api_security_platform
  (Verify credentials in .env)

Module Not Found
  Backend: npm install
  Scanner: pip install -r requirements.txt (with venv activated)

Scanner Not Responding
  $ curl http://localhost:8000/health
  Check logs: docker-compose logs scanner

Invalid Postman Collection
  $ python -m json.tool collection.json
  (Validate JSON structure)
